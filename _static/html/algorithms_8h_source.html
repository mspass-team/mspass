12 /* \brief Apply agc operator to three component seismogram data.
13 
14  Automatic gain control (agc) is a standard operation in seismic
15 reflection processing. The algorithm used her is a variant of that in
16 seismic unix but applied to vector data. That is scaling is no
17 determined by absolute value of each sample but th vector amplitude of
18 each sample. Scaling is determined by the average vector amplitude
19 over a specified time window length. There isa ramp in and ramp off
20 range of size equal to the window length. agc was notorious in the
21 early days of seismic processing for making it impossible to recover
22 true amplitude. We remove that problem here by returning a TimeSeries
23 object whose contents contain the gain applied to each sample of the
24 original data.
25 
26 \param d - data to apply the operator to. Note it is altered.
27 \param twin - length of the agc operator in seconds
28 
29 \return TimeSeries object with the same number of samples as d. The
30  value of each sample is the gain applied at the comparable sample in d.
31 
32 This function does not throw an exception, but can post errors to the
33 ErrorLogger object that is a member of Seismogram.
34 */
