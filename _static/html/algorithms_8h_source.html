11 /* \brief Apply agc operator to three component seismogram data.
12 
13  Automatic gain control (agc) is a standard operation in seismic
14 reflection processing. The algorithm used her is a variant of that in
15 seismic unix but applied to vector data. That is scaling is no
16 determined by absolute value of each sample but th vector amplitude of
17 each sample. Scaling is determined by the average vector amplitude
18 over a specified time window length. There isa ramp in and ramp off
19 range of size equal to the window length. agc was notorious in the
20 early days of seismic processing for making it impossible to recover
21 true amplitude. We remove that problem here by returning a TimeSeries
22 object whose contents contain the gain applied to each sample of the
23 original data.
24 
25 \param d - data to apply the operator to. Note it is altered.
26 \param twin - length of the agc operator in seconds
27 
28 \return TimeSeries object with the same number of samples as d. The
29  value of each sample is the gain applied at the comparable sample in d.
30 
31 This function does not throw an exception, but can post errors to the
32 ErrorLogger object that is a member of Seismogram.
33 */
