double mspass::seismic::PowerSpectrum::amplitude ( const double  f) const

Amplitude at a given frequency.

This is an overloaded method that returns the interpolated amplitude (sqrt(power)) at a requested frequency. If the frequency exceeds the Nyquist the function silently returns 0.
