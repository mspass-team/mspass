Detailed Description

Frequency-domain generalized iterative deconvolution.

This engine shares the sparse GID iteration semantics of TimeDomainGIDDecon but applies the candidate inverse operator in the frequency domain. Processing requires both a loaded signal window and residual-domain noise window. External PowerSpectrum noise is only an ns_gid inverse-operator regularization estimate; it does not replace load(d, dwin, nwin) or loadnoise(d, nwin).
